TETRIS OVERLOAD README
before running the Tetris Overload game, make sure the music files are downloaded
 - 'tetrisb.mid'
 - 'tetrisc.mid'
if they are not, download them here
 - http://invpy.com/tetrisb.mid
 - http://invpy.com/tetrisc.mid

to run the game, open up main.py and run it

to compare the differences side-by-side from the original code, copy and paste the text in main.py into this website
 - https://inventwithpython.com/pygame/diff/?p=tetromino#diff
